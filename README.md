Used in the Astralis version requester

- Performance Optimized
- Third Person Player Spectating (yourself or others)
- First Person (yourself or others) - (Cosmetic visibility disabled if not yourself)
- Second Person Player Spectating (yourself or others)
- Orbital player spectating (Broken)
- Smart Casting
- Auto Casting (Unreliable and buggy)
- Multple Spectating Smoothing Modes (if smoothing is enabled)
- Dynamic Field Of View Based On Spectated Player Speed (Unreliable if used with high positional camera smoothing)
- Free Camera
- Rig Body And Arm Lerping Change
- Player Nametags
- Player Platform Checking
- Player FPS Checking
- Player FPS Spoofing Notifier
- Player Estimated Ping Checking

- Notifications
- Smooth GUI Usage And Modern Design
- Microphone Management (push to talk via key or GUI button toggle)
- Welcome Modal (Independently Saved Data)
- Tutorial (Independently Saved Data)
- Room Leaderboard
- Tag Feed
- Sound Feed (Buggy and unreliable after recent game updates)
- Closest Tagged Player To Current Spectated Player

- Move Game Audio To Camera Position
- Basic AntiCheat Attempting Gameplay Vizualizer (Unreliable)
- Manual Performance Manager (Buggy)
- Advanced Preset System (May be buggy under certain conditions but tools are provided to manually manage or fix preset issues via folder through the preset section tab)
- Ingame Camera (Currently not being worked on as everything works besides a couple bugs resulting in it being unable to spawn)
- Basic/Essentials Photon And Room Manager
- Local Player And Client Settings
- Player Muting
- GUI And Nametag Font Settings (Buggy with certain fonts)
- Time And Basic Weather Manager
- Full Extensive Map Loader (May cause a gorilla tag anticheat flag or result in a ban if too many maps are loaded at once)

- BepInEx Plugin Management (will be added back in later builds)
